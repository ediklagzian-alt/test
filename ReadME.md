# Vadim Bank

Vadim Bank — это больше чем финансы. Это символ уверенности для тех, кто ценит стабильность и индивидуальный подход. 
Мы объединяем классические традиции банковского дела с гибкостью современных технологий, чтобы ваш капитал работал так же надежно, как само имя `Vadim`

## Что реализовано

- клиенты банка (верифицированные и сомнительные)
- три типа счетов: дебетовый, депозитный, кредитный
- операции: пополнение, снятие, перевод
- отмена транзакций (последней операции)
- лимит для сомнительных клиентов на снятие и перевод
- проверка ключевых бизнес-правил (срок депозита, кредитный лимит, положительные суммы)

## Паттерны

- `Builder` - пошаговое создание клиента (`ClientBuilder`)
- `Factory Method` - создание счетов (`AccountFactory`)
- `Command` - транзакции и их откат (`WithdrawCommand`, `TransferCommand`)

## Как запустить

Требования: Python 3.10 и выше.
Лимит для сомнительных клиентов настраивается через переменную `BANK_SUSPICIOUS_LIMIT`.

```bash
cd src
python main.py
```

## Быстрый сценарий проверки

1. Создай клиента без адреса/паспорта (сомнительный клиент).
2. Открой ему счет и пополни баланс.
3. Попробуй снять сумму больше лимита банка - операция будет отклонена.
4. Сделай перевод между счетами.
5. Выполни пункт меню "Отменить последнюю транзакцию".
6. Проверь, что баланс вернулся к предыдущему состоянию.
